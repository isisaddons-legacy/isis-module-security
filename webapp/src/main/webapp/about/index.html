Apache Isis™ is a framework to let you rapidly develop domain-driven apps in Java. This example app demonstrates the Isis Addons' Security Module, configured for a very simple and purposefully minimal application.

To access the app:

  • wicket/

    provides accesses to a generic UI for end-users and administrators.

  • restful/

    provides access to a RESTful API conformant with the Restful Objects spec.

The default user/password is isis-module-security-admin/pass.
